# 🌐 Voice Translator

A modern, responsive, and premium web application for translating text and speech between English and various Indian languages in real-time. Features browser-native speech recognition and text-to-speech output.

## 🚀 Live App (24/7)
The application is hosted live on Render and can be accessed on both desktop and mobile devices:

### 👉 **[https://multilanguage-ai.onrender.com](https://multilanguage-ai.onrender.com)**

---

## 🛠️ Features

* **Voice Input (Speech Recognition):** Directly capture speech in multiple Indian languages (Hindi, Bengali, Tamil, etc.) using the browser's native speech recognition APIs.
* **Text-to-Speech Output:** Speak translations out loud using high-quality native synthesis voices.
* **Responsive Dark Mode:** Polished glassmorphic card design optimized for mobile phone screens and tablets.
* **Multi-Language Support:** Translates between English and Hindi, Marathi, Bengali, Gujarati, Tamil, Telugu, Kannada, Malayalam, Punjabi, and Urdu.

---

## 💻 Local Running Instructions

If you want to run this application locally on your computer:

1. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Start the local server:
   ```bash
   python app.py
   ```
3. Open your browser and navigate to `http://localhost:5000`.
